# 🎵 RythmHub

RythmHub is a modern music discovery web application inspired by popular music streaming platforms. It allows users to search for songs, explore trending music, play song previews, and save their favorite tracks through a clean and responsive interface.

## ✨ Features

- 🔍 Search songs, artists, and albums
- 🎵 Display trending songs on page load
- ▶️ Play song previews
- ❤️ Save favorite songs using Local Storage
- 🕒 Recently played songs
- 🌙 Dark mode
- 📱 Fully responsive design
- ⚡ Fast and interactive UI
- 🔄 API Integration using Fetch API & Async/Await

## 🛠️ Tech Stack

- HTML5
- Tailwind CSS
- JavaScript (ES6)
- Fetch API
- Local Storage

## 🚀 Future Enhancements

- 👤 User authentication
- 🎼 Custom playlists
- 📝 Lyrics support
- 🎧 Genre-based recommendations
- 🎵 Full music streaming
